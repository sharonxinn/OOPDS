3. The two common programming methods in practice today are:

Procedural and Object-Oriented.

4. _____ programming is centered around functions, whereas ______ programming is centered around objects.

Procedural programming is centered around functions, whereas Object-Oriented programming is centered around objects.

5. An object is a software entity that combines both ____ and ____ in a single unit.

Data and methods (or functions).
